# Buy Bee - E-Commerce Platform

Buy Bee is a modern, high-performance e-commerce platform built with Next.js, offering a seamless and responsive shopping experience. From browsing trending products to managing your cart, Buy Bee is designed for speed and style.

## üöÄ Features

- **Rich Homepage**: Features dynamic sections including Hero sliders, Top Brands, Best Selling items, and New Arrivals.
- **Enhanced Product Details**: 
  - **Dynamic Interactivity**: Interactive quantity and color selectors.
  - **Ratings Summary**: Visual star breakdown and average rating display.
  - **Verified Reviews**: Detailed customer reviews with verified badges.
  - **Clean Specs**: Organized key specifications display for every product.
- **Advanced Cart System**: Real-time quantity adjustments and price updates directly in the cart.
- **Product Discovery**: Category-based filtering and smart sorting to find what you need quickly.
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop viewing.

## üì¶ Project Setup

Follow these steps to get the project running locally:

### 1. Clone the repository
```bash
git clone <repository-url>
cd buy-bee-scaleup
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the development server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.

## üß© Folder Structure Highlights

- `src/app`: Next.js App Router pages and layouts.
- `src/components/pages`: Feature-specific components organized by page (e.g., `detailsPage`, `home`).
- `src/components/ui`: Reusable UI primitives (Buttons, Selects, etc.).
- `src/data`: Centralized product and application data.
- `src/types`: TypeScript interface and type definitions.
